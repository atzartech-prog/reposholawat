# Repository Teks Qosidah & Sholawat Nabi - Thobiby Qolby App

Aplikasi web interaktif, responsif, dan premium untuk mempelajari serta mengamalkan berbagai teks Qosidah dan Sholawat Nabi. Data lirik dan kisah latar belakang bersumber dari blog **[Pecinta Habibana (Thobiby Qolby)](https://pecintahabibana.wordpress.com/teks-qosidah/)**.

Aplikasi ini dibangun menggunakan **HTML, Vanilla CSS (CSS Variables untuk Light/Dark mode), dan JavaScript murni** dengan desain estetik bernuansa hijau botol dan aksen emas islami (premium glassmorphic interface).

---

## 🌟 Fitur Utama

1. **Pencarian Cepat & Filter Kategori:** Cari lirik sholawat secara instan berdasarkan judul, cuplikan bahasa Arab, transliterasi Latin, maupun terjemahan bahasa Indonesia. Filter berdasarkan kategori: *Populer, Sholawat, Qosidah, Maulid, atau Koleksi Favorit*.
2. **Koleksi Favorit (Bookmark):** Simpan sholawat yang sedang Anda hafalkan atau amalkan sehari-hari ke daftar favorit Anda. Data otomatis tersimpan di peramban menggunakan *LocalStorage*.
3. **Kustomisasi Membaca (Reading Mode):**
   - Mengatur ukuran font tulisan Arab (A-, A, A+, A++) demi kenyamanan mata.
   - Menyalakan/mematikan teks Latin (Transliterasi) jika Anda ingin fokus membaca huruf Arab.
   - Menyalakan/mematikan teks Terjemahan bahasa Indonesia.
4. **Salin Lirik Mudah (Copy Utility):** Fitur sekali klik untuk menyalin seluruh lirik beserta teks Arab, latin, terjemahan, dan rujukan sumbernya untuk dibagikan ke media sosial atau aplikasi pesan.
5. **Akses Audio/Video Instan:** Tombol khusus yang menghubungkan Anda langsung ke pencarian YouTube untuk mendengarkan pelafalan/nada sholawat secara langsung.
6. **Latar Belakang & Fadhilah:** Penjelasan sejarah atau manfaat masing-masing sholawat untuk memperdalam penghayatan saat mengamalkannya.
7. **Desain Premium Modern:** Dilengkapi dengan mode Gelap (Dark Mode) dan Terang (Light Mode) serta adaptif penuh untuk perangkat mobile/smartphone.

## 💡 Cara Menjalankan Aplikasi

Aplikasi ini berjalan sepenuhnya di sisi klien (*client-side*). Anda tidak memerlukan server khusus atau instalasi Node.js/framework tambahan.

1. Unduh atau klon repositori ini.
2. Buka folder `reposholawat`.
3. Klik ganda file `index.html` untuk langsung membukanya di browser Anda (Chrome, Firefox, Safari, Edge, dll.).
4. Untuk penggunaan di server lokal (opsional), Anda bisa menggunakan ekstensi *Live Server* di VS Code atau perintah terminal:
   ```bash
   npx serve .
   ```
